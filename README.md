# image-steganography-lsb
# Image Steganography project using LSB Encoding and Decoding in C


## Overview
This project implements Image Steganography using the Least Significant Bit (LSB) technique in C. The system hides secret text data inside an image without significantly affecting image quality. The hidden message can later be extracted using the decoding process.

## Features
- Encode secret message into image
- Decode hidden message from image
- LSB-based data hiding technique
- Command-line implementation
- Supports BMP image format

## Technologies Used
- C Programming
- File Handling
- Bitwise Operations
- Image Processing Concepts

## Project Workflow
1. Read source image
2. Encode secret message into image pixels
3. Generate stego image
4. Decode hidden message from stego image

## Applications
- Secure communication
- Data hiding
- Digital watermarking
- Information security

## Output
- Encoded image containing hidden data
- Decoded secret message
